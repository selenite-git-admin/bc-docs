# Port 4200 for bc-sdg OData server (D014 dev tools range)

## Context

Follows D014 port reservation convention. Single port with path-based landscape routing avoids port sprawl.

## Decision

bc-sdg OData server runs on port 4200, within the dev tools range (4000-4099 → extended to 4200 for SDG). Each simulated landscape gets a sub-port or path prefix: /s4cloud, /s4onprem, /ecc — all on port 4200. BareCount connections point to http://localhost:4200/s4cloud/sap/opu/odata4/sapi/... etc.
