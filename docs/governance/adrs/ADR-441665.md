# NPM supply chain mitigation via AWS CodeArtifact

## Context

Mitigates supply chain risk (RSK-cb8929): npm registry outage, package yanking (left-pad), dependency compromise (event-stream). CodeArtifact caches every version pulled — cached copies survive upstream removals. Aligns with ISO 27001 A.14.2.1 (secure development) and A.12.6 (vulnerability management). Tier 2 approach: low effort, high protection against availability + yanking risks.

## Decision

All BareCount npm installs route through AWS CodeArtifact (domain: barecount, repo: npm-mirror → npm-public → npmjs.org, region: ap-south-1). Project-level .npmrc files committed to Git enforce the registry. Auth token refreshed via aws codeartifact login (12h TTL). No direct npmjs.org usage.
