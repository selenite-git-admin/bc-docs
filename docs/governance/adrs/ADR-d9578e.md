# Dashboard route slugs — keep short

## Context

Shorter URLs are cleaner for bookmarks and sharing. The dashboard display name already provides full context. Keeps URLs concise.

## Decision

Align dashboard route IDs to match the short URL slugs (e.g., id 'quota' not 'quota-attainment') rather than lengthening URLs. IDs adapt to slugs.
