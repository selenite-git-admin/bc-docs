# Source Catalog: Admissions → Source Contracts (extraction + source + admission)

## Context

Superseded by D068. Source Contracts nav item removed entirely — contracts surface from Reader detail page.

## Decision

Rename "Admissions" nav item in Source Catalog to "Source Contracts". The page shows all 3 source-chain contract types (extraction, source, admission) filterable by category. This ensures extraction and source contracts have a management surface — previously only admission contracts were visible. Route changed from /registry/contracts/admission to /registry/contracts/source-chain (old URL redirects). ContractDetailPage already handles version management and JSON view for any contract type.
