# BCF code layout — consolidate the bcf-* module family under src/registry/bcf/

## Context

src/registry had grown to 25 sibling directories, 9 of them a homogeneous bcf-* family (5-9 files each, one shared pattern: recommendation/adjudication writers over PanelOutputRecordService) — the prefix was doing a folder's job and every new recommendation kind added top-level sprawl. Sequenced deliberately: after the vocabulary-rename sweep landed (verified: src/registry has zero uncommitted modifications) and after the D483 in-process panel accumulated live proof (7 runs incl. the full admission→activation loop). Operator GO 2026-07-04; TSK-32e427.

## Decision

bc-core's BCF workflow modules consolidate under one root: src/registry/bcf/ with panel/ (← registry-authoring-panel: B6 run path, in-process trio, prompts, publication, provenance), recommendations/<kind>/ (← the seven bcf-*-recommendation modules with the bcf- prefix and -recommendation suffix dropped from folder names), adjudication/ (← bcf-panel-operator-adjudication), phase3-orphan-repair/ (← bcf-phase3-orphan-repair), and telemetry/ (already in place). Folder moves only — file names, module classes, routes, and behavior are unchanged (pure mechanical relocation, git mv for rename tracking).

Explicitly OUT of scope (peers BCF governs or consumes, not BCF workflow): concept-registry/ (the vocabulary substrate), framework-approval/ (C5), framework-calibration/, d443-phase2-cert-supply/, dto/, and the root-level panel-output-record service trio. Widening the umbrella is a later decision, not this one.

Same-commit companions mandated by the move audit: (1) nest-cli.json gains the asset include for registry/bcf/panel/prompts/**/*.md — the BCF v1.3 prompt files were NEVER in the dist asset list (latent dist-only failure; the src fallback masked it in local dev); (2) the in-process prompt-loader's literal dist→src fallback path updates to the new location. Verification bar: tsc clean on moved files, full vitest at the pre-existing baseline, zero surviving old-path references, one live panel smoke through the moved code.
