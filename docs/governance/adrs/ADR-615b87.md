# Intervention contract — 6th contract family in platform contract schema

## Context

Intervention contracts follow the same governance lifecycle as other contracts. Platform owns the rules, tenants customize via contract_binding. Keeping in tenant DB would require cross-DB merging.

## Decision

Created contract.intervention_contract + approval + version as the 6th contract family. Governs the intervention evaluation boundary: clock duration, escalation policy, accountability, evaluation rules, trigger conditions (future). Replaced tenant-level intervention_definition. Action = Intervention (same boundary, intervention is the correct vocabulary).
