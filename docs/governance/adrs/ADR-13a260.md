# Frontend SDK: amazon-cognito-identity-js (not Amplify)

## Context

Lightweight (~45KB vs ~150KB). Full control over UI — works with existing custom auth screens (LoginPage, MfaVerificationPage, ForgotPasswordPage). No Amplify config layer (aws-exports.js) needed. Direct Cognito API calls: authenticateUser, sendMFACode, forgotPassword, completeNewPasswordChallenge.

## Decision

Use amazon-cognito-identity-js for frontend Cognito integration. Not @aws-amplify/auth.
