# QA Shift-Left — Coding Standards in CLAUDE.md

## Context

Sessions repeatedly generated code that failed QA audits, requiring fix-up sessions. By declaring the rules upfront, sessions will produce compliant code from the start, reducing QA findings to near-zero and eliminating rework cycles.

## Decision

Embed all 12 bc-qa audit rules as proactive coding directives in CLAUDE.md so Claude sessions enforce them during code generation, not just detect violations post-hoc.
