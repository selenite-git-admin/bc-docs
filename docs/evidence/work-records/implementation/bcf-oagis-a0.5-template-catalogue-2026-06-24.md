## 1. Cat-1 cluster review groups — per-characteristic shape matrix (PIN AUTHORITY)

Six clusters group proposed new characteristics by inferred `representation_term`. Within each cluster, the per-characteristic shape matrix is the operator-pin authority. The matrix carries: bf_name, distinct shape tuple(s), shape_conflict flag, OAGIS sample, target slices, used_by_bc_target_count, execution class.

**Schema for each Cat-1 matrix row:**

| Field | Meaning |
|---|---|
| `bf_name` | OAGIS bf_name slug (the candidate term root) |
| `(rep_term \| data_type \| semantic_role)` | Distinct shape tuple(s) inferred from OAGIS; multiple shapes = shape_conflict |
| `shape_conflict` | TRUE if multiple distinct shape tuples; operator must split or pick |
| `OAGIS sample` | One representative (noun.component.field) for citation seed |
| `used_by_bc_target_count` | Number of BC ledger rows referencing this characteristic |
| `target slices` | Entity slices where the characteristic will bind |
| `execution class` | AMBER for clean admissions; AMBER-with-conflict for shape conflicts |
